/getWholeSet
  • Methods: GET, HEAD
  • Query Params: None
  • Body Params: None
  • Returns: All of the data from the set in JSON. Displaying Cards may take a while due to size!
  • Example: [app-link]/getWholeSet

/getCardByName
  • Methods: GET, HEAD
  • Query Params: searchTerm
  • Body Params: None
  • Returns: All of the cards that match the search terms (comma seperated) in the name
  • Example: [app-link]/getCardByName?searchTerm=valgavoth

/getCardByKeyword
  • Methods: GET, HEAD
  • Query Params: searchTerm
  • Body Params: None
  • Returns: All of the cards that match the keywords (comma seperated) in the card
  • Example: [app-link]/getCardByName?searchTerm=trample,flying

/getRandomBooster
  • Methods: GET, HEAD
  • Query Params: None
  • Body Params: None
  • Returns: A randomly selected Booster Pack from the set - replacing special guest cards with Rare/Mythic cards.
  • Example: [app-link]/getRandomBooster

/addBooster
  • Methods: POST
  • Query Params: None
  • Body Params: boosterString
  • Returns: All of the cards that match the keywords (comma seperated) in the card
  • Example: A POST request with a body parameter of boosterString: "250:c10l2" would have a 250 weight of getting chosen, the booster pack having 10 common cards and 2 land cards.

/addCard
  • Methods: POST
  • Query Params: None
  • Body Params: cardName, manaCost, typeLine, cardText
  • Returns: All of the cards that match the keywords (comma seperated) in the card
  • Example: a POST request with body paramteres of cardName: "A New Card", manaCost: "WWWB6", typeLine: "Enchantment Creature", and cardText: "This is a new card, it's very good!", would add a card with that information if a card by the name of "A New Card" didn't exist in the set already.
